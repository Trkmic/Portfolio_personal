Portfolio profesional desarrollado con un enfoque en **Desarrollo Backend**, utilizando una estética de terminal "Cyberpunk/Hacker" optimizada para destacar mis proyectos y formación académica.

## 🚀 Vista Previa
> **🔗 [Ver Portfolio en Vivo](https://trkmic.vercel.app/)**

## 🛠️ Tecnologías Utilizadas

* **Astro 5.0**: Framework web orientado al rendimiento.
* **Tailwind CSS**: Estilizado basado en utilidades para el diseño "Terminal Dark".
* **FontAwesome & DevIcons**: Iconografía para redes y stack tecnológico.
* **TypeScript**: Desarrollo robusto y tipado.

## 🎨 Estética y Diseño
El sitio utiliza una paleta de colores basada en **Negro Puro (#000000)** y **Verde Esmeralda (#10b981)**, simulando una interfaz de consola de comandos. Se aplicaron efectos de resplandor neón mediante sombras externas y tipografía monoespaciada para reforzar la identidad de **Desarrollador Backend**.
